基于Java语言和MySQL数据库。开发工具使用IDEA，基于SSM框架。
用户权限分为普通用户和管理员。普通用户可以申请会议室使用，对自己申请的会议室进行管理。
管理员可以管理会议室(包括会议室信息的增删改查)，也可以管理用户信息。 
已存放sql结构，导入即可。
